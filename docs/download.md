## Download

RedStoneMe is distributed as a Bukkit plugin JAR.

### Project Pages

- [GitHub Releases](https://github.com/Stellionix/RedStoneMe/releases)
- [CurseForge](https://www.curseforge.com/minecraft/bukkit-plugins/redstoneme)
- [BukkitDev](https://dev.bukkit.org/projects/redstoneme)

### Build It Yourself

If you want to build from source:

```bash
./gradlew shadowJar
```

The packaged plugin JAR is generated in:

```text
build/libs/redstone-me-<version>.jar
```

Use `./gradlew printVersion` to print the current project version in CI or release scripts.
